# Google Maps URL Generator

This project allows users to input a house number and get a Google Maps URL for the corresponding location.

## How to Use
1. Open the `index.html` file in a browser.
2. Enter the house number in the input field.
3. Click the "ค้นหา" button to get the Google Maps URL.

## Deployment
You can deploy this project using GitHub Pages:
1. Push the project to a GitHub repository.
2. Go to the repository's **Settings** > **Pages**.
3. Select the branch and directory (e.g., `main` and `/root`).
4. Save the settings and get the URL for your hosted project.
